# OneNote 页面内标题导航目录 — 安装与使用说明

## 方案说明

本方案基于 **Onetastic** 宏平台，将导航目录功能直接集成到 OneNote 工具栏中。安装后，在 OneNote 的 **"开始"** 选项卡中会出现宏按钮，一键即可生成页面内标题导航目录。

## 第一步：安装 Onetastic

1. 打开 OneNote 桌面版
2. 访问 Onetastic 官网：**https://www.getonetastic.com/download**
3. 下载与您的 Office 位数匹配的版本（通常是 64 位）
   - 如果不确定，在 OneNote 中点击 **文件 → 账户 → 关于 OneNote**，查看位数
4. 运行下载的安装程序，按提示完成安装
5. **重启 OneNote**
6. 安装成功后，OneNote 的 **"开始"** 选项卡中会出现 Onetastic 的按钮组

> Onetastic 是免费的。基础功能（包括运行宏）完全免费。

## 第二步：安装宏

### 通过官方macroland安装

1. 打开 OneNote
2. 点击 **"开始"** 选项卡 → **"Download Macros"**（下载宏）
3. 在弹出窗口中，搜索 **"页面内标题导航目录"** 
4. 选择 **"页面内标题导航目录"** ，点击 **"Install"**（安装）
5. 安装成功后，便可在工具栏中使用

## 第三步：使用

### 生成导航目录

1. 打开一个包含标题的 OneNote 页面
2. 确保页面中的标题使用了 OneNote 内置的**标题样式**（开始 → 样式 → 标题1/2/3...）
3. 点击 **"开始"** 选项卡 → 点击 **"TOC"** 按钮下的 **"页面内标题导航目录"** 宏按钮
4. 页面顶部会出现带超链接的标题目录

### 宏菜单选项

| 选项             | 功能                                         |
| ---------------- | -------------------------------------------- |
| **生成导航目录** | 扫描标题，在页面顶部生成可点击的导航目录     |
| **更新导航目录** | 删除旧目录，重新生成（页面标题有变化时使用） |
| **删除导航目录** | 删除页面顶部的导航目录                       |

### 跳转到标题

- 点击目录中的标题文字，即可跳转到页面中对应的位置
- 每个标题前有层级符号，方便区分级别

### 目录效果示例

```
📑 页面导航目录（共 6 个标题）

━ 第一章 概述
┃ 1.1 背景
┃ 1.2 目标
┗ 1.3 范围
  · 详细说明
  ○ 子项内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 设置快捷键（可选，需要pro才能使用）

1. 点击 **"开始"** → 找到 **"Onetastic"** 功能区下的 **"Tools and Settings"**
2. 点击 **"Onetastic Options"**（选项）
4. 在 **"Shortcuts"** 中设置快捷键（如 `Ctrl+Shift+C`）
5. 点击 **"OK"** 保存

## 常见问题

| 问题                     | 解决方案                                        |
| ------------------------ | ----------------------------------------------- |
| 找不到 Onetastic 按钮     | 确认已安装 Onetastic 并重启 OneNote             |
| 当前页面中没有找到标题     | 确保使用了 OneNote 内置的标题样式（非手动加粗） |
| 点击目录标题没有跳转       | 确认标题链接生成正确，尝试重新生成目录          |
| 宏按钮灰色不可用           | 确认当前有一个打开的页面                        |

## 注意事项

1. **标题必须使用内置样式**：仅通过加粗、放大字号等方式设置的"标题"不会被识别
2. **链接在本机有效**：生成的跳转链接基于 OneNote 内部段落 ID，在不同电脑上可能失效
3. **更新目录**：页面标题有增减变化时，再次运行"页面内标题导航目录"即可刷新
4. **兼容性**：Onetastic 仅支持 Windows 桌面版 OneNote，不支持 Mac/网页版
